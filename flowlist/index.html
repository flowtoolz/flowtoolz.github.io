Flowlist

Capture ideas quickly. Organize your stuff freely.
Focus on what's important.

Innovative

A unified design offers flexibility
as well as simplicity.

The Familiar

Like other task managers, Flowlist shows 3 main views: A task list in the middle, an overview on the left for context, and a detail view on the right showing details like subtasks of the selected task.

More Flexible

Unlike other task managers, Flowlist does not force your stuff into fixed categories like Project or Subtask. In Flowlist, tasks may contain tasks, and that's it. You can create any hierarchy of tasks.

Yet More Elegant

In Flowlist, the 3 main views are all task lists. Each shows the details of what is selected in the list to its left. Just hit the left or right Arrow Key to slide the context or the detail view into focus.

Quick

Intuitive keyboard commands
enable rapid editing and a clean interface.

Navigate

Use ↑ or ↓ to select a different task. You can go up and down within a list. You can also move the focus left to the overview list or right to the subtasks of the currently selected task.

Add and Delete

Hit Space to add a task to the top of the list. Or use ↵ and ⌫ to write and delete tasks just like lines or words in a text.

Prioritize

Use ⌘↑ or ⌘↓, to move a single selected task up or down in the list. Thereby you can visually prioritize tasks. You can also move completed tasks that were put to the end of the list.

Do Batch Processing

Some actions can be applied to multiple tasks at once. To select multiple tasks, use ⇧↑ and ⇧↓. Or hold ⌘ while clicking a task to add or remove it from the selection.

Start and Pause

Hit ⌘→ to set tasks in progress or to pause them. Tasks in progress slide to the top. Paused tasks slide back down.

Check Off

Click its check box to complete a task. Or hit ⌘← to complete the first active selected task. When completing tasks they move below the last active task, so you'll see them sorted by order of completion.

Rename

Hit ⌘↵ to rename the first selected task. If multiple tasks are selected, entering the name for one task immediately unselects that task and begins editing the name of the next selected task.

Group

Select multiple tasks and hit ↵ to group the selection and name the group. Tasks that contain others as subtasks have an arrow indicator. If you select such a task, the detail list on the right shows the subtasks.

Add Subtasks

If a task has no subtasks, you can still focus its subtask list and start adding tasks there. The group indicator in the overview list on the left appears/disappears when the focused list becomes non-empty/empty.
