The Familiar

Like other task managers, Flowlist shows 3 main views: A task list in the middle, an overview on the left for context, and a detail view on the right showing details like subtasks of the selected task.

More Flexible

Unlike other task managers, Flowlist does not force your stuff into fixed categories like Project or Subtask. In Flowlist, items may contain items, and that's it. You can create any hierarchy of items.

Yet More Elegant

In Flowlist, the 3 main views are all item lists. Each shows the details of what is selected in the list to its left. Just hit the left or right arrow key to slide the context or the detail view into focus.

Quick

Intuitive key commands
allow for rapid editing and a clean interface.

Navigate

Go up ↑ or down ↓ within the currently focused list. Focus details → or get an overview ←.

Add & Delete

Add an item to the top of the list Space or add ↵ and remove ⌫ items just like lines in a text.

Prioritize

Move an item up ⌘↑ or down ⌘↓ to visually prioritize within the currently focused list.

Do Batch Processing

Extend the selection up ⇧↑ or down ⇧↓. Add/remove items to/from the selection ⌘Click.

Start & Pause

Start or pause an item's progress ⌘→. It will slide up or down to adjust its priority.

Check & Uncheck

Check or uncheck an item ⌘←. It will slide up or down accordingly. Selection will auto-adjust.

Rename

Rename the first selected item ⌘↵. Finish typing ↵ to rename the next selected item.

Group

Group items together ↵ and enter a title. The initial group state reflects the grouped items.

Add a Description

Focus an empty item → and add subitems into it ↵ or to the top of its subitem list Space.

Duplicate & Move

Duplicate items or move them into a different item via copy ⌘C, cut ⌘X and paste ⌘V.

Undo

Paste the last deleted items ⌘Z in case you accidentally deleted items from the focused list.

Import

Copy content ⌘C from any document or web page and paste it as distinct items ⌘V into a list.
