All Languages

Add support for any programming language
by simply providing the corresponding LSP server.

1. Get LSPService

Download LSPService. It's a free open-source webservice that runs locally and enables Codeface to understand any codebase.

2. Add Language Support

Add language support if you need more than Swift. LSPService just needs to know how to launch an LSP server for a given language.

3. Restart Codeface

Launch LSPService after it's configured. Then restart Codeface. Codeface automatically detects- and talks to LSPService.
